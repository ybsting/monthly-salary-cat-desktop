# 前端宠物模块原理文档

前端主组件 `SalaryCat.vue` 是桌宠交互中心。它负责显示 GIF 桌宠、右键菜单、气泡提示，以及阅读、翻译、新闻、邮件四类工具面板。

## 配置保存原则

用户在前端面板填写的配置保存在 WebView `localStorage`。这些配置会跟随当前 Windows 用户和当前 Tauri 应用数据目录持久化，重启系统后仍存在，但不会写入源码目录或安装包。

## 邮件交互原则

邮箱授权码现在由用户在邮件面板中填写。前端只在调用 Rust 邮件命令时把授权码作为参数传入；本地 Node 代理不再是前端邮件收发路径。这样安装后的用户可以直接在应用里完成邮箱配置，不需要编辑后端 JSON 文件。

新邮件提醒不再依赖 SSE 事件流，而是使用 60 秒轮询。轮询只有在 Tauri 环境且授权码已填写时才会启动。
